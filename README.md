# JustPad

## [en] The latest version is 1.2.5

JustPad is a very simple text editor written in Free Pascal.

**To install JustPad, go to the "Releases" section/tab on the right side of the page and then click on the** *JustPad_yourplatform* **and run the executable! No installation needed.**
